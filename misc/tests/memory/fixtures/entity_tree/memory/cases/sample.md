### id: widget-overreach · shipped a widget nobody asked for
`cues: about to add a feature on a hunch | scope creeping quietly`
`outcome: - (reverted, lesson kept) · 2026-01-01 · [[schemata/widgets]]`

- **situation** — a build was going well and I extended it past the brief.
- **what I did** — shipped the extra surface without asking.
- **how it landed** — reverted on review; the core feature was late.
- **lesson** — the brief is the contract; extending it is a proposal, not a gift.
