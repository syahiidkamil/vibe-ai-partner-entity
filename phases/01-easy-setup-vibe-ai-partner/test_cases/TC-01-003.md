# TC-01-003: Setup with Language Packs — Japanese + Chinese

## Type
happy-path

## Preconditions
- Fresh clone or no prior language downloads
- uv installed

## Steps
1. Run `uv run vibe setup`
2. Select engine: Kokoro ONNX
3. When prompted for languages, select Japanese and Chinese
4. Wait for all downloads

## Expected Result
- Language options displayed with sizes:
  - English (included, 0 MB)
  - Japanese (+526 MB UniDic)
  - Chinese (+305 MB separate model)
  - French, Spanish, etc. (0 MB, espeak-based)
- Each download shows individual progress bar
- UniDic downloads for Japanese
- Chinese model (kokoro-v1.1-zh.onnx + voices + config) downloads
- Total download size shown before confirmation
- All models cached correctly

## Priority
high
