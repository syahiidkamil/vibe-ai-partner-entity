# TC-01-001: Fresh Setup — Kokoro ONNX with English only

## Type
smoke

## Preconditions
- Fresh clone of the repo (no prior setup)
- uv installed
- No cached models in `~/.cache/kokoro-onnx/`

## Steps
1. Run `uv run vibe setup`
2. Select engine: Kokoro ONNX (option 1)
3. When prompted for languages, skip additional languages (press Enter/N)
4. Wait for model download to complete

## Expected Result
- Prerequisite checks pass (Python version, uv version, disk space)
- Engine selection shows 3 options with sizes
- Python deps install via `uv sync --extra kokoro-onnx`
- Model downloads show progress bar with speed and percentage
- `config.json` updated with `"ttsEngine": "kokoro-onnx"`
- Models cached at `~/.cache/kokoro-onnx/`
- Clear "next steps" message printed

## Priority
critical
