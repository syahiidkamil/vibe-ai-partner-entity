# TC-01-007: Port Already in Use

## Type
edge-case

## Preconditions
- Another process running on port 5111

## Steps
1. Start something on port 5111 (e.g., `python -m http.server 5111`)
2. Run `uv run vibe start`

## Expected Result
- Clear error: "Port 5111 is already in use"
- Suggests: "Run 'uv run vibe stop' or use '--port 5112'"
- Does not crash with raw socket error

## Priority
medium
