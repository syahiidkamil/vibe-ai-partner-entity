# TC-01-010: Download Interrupted — Resume/Retry

## Type
edge-case

## Preconditions
- Setup in progress, model downloading

## Steps
1. Run `uv run vibe setup`, select Kokoro ONNX
2. During model download, press Ctrl+C to interrupt
3. Run `uv run vibe setup` again, select same engine

## Expected Result
- Interrupted download leaves a `.part` file (not a corrupt full file)
- Re-running setup detects incomplete download
- Download resumes or restarts cleanly
- No corrupt model files in cache

## Priority
high
