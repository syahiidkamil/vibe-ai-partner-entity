# Equipment (MCGG S6)

Items heroes carry, up to **3 per hero** (3 gear slots; Magic Crystals are separate and take no
slot). Each item has a **Basic** and an **Enhanced** form; enhancing raises stats and/or adds a
Unique Passive. Items drop from creep (PvE) rounds and the Go Go Box, not a gold shop. Drops and
economy: [[equipment_and_economy]].

## The tabs (one file per UI tab)

| Tab | Items | File |
|---|---|---|
| Regular | 28 (5 Basic actives + 23 stat items) | [regular](regular.md) |
| Magic Crystal | 16 | [magic_crystal](magic_crystal.md) |
| Synergy Exclusive | ~17 (4 captured) | [synergy_exclusive](synergy_exclusive.md) |
| Commander Exclusive | 15 | [commander_exclusive](commander_exclusive.md) |
| Special | ~4 (2 captured) | [special](special.md) |

Synergy Exclusive and Special are only partly captured; the rest need more scroll shots. Names,
stats, and Unique-Passive values are first-party from the in-game UI (Basic + Enhanced detail
panels). Exact per-id stat rows + the name↔id keying: [source_map](source_map.md).

## Type groups (Regular)
Items sort under **Basic / Physical / Magic / Defense / Hybrid ATK**. Hybrid items grant Physical
and Magic at the same value (the UI's "Hybrid ATK" / "Hybrid DEF").

## Two key cross-links
- **Magic Crystals** grant a synergy (one per synergy that has one); the in-battle source of the
  +1 synergy-count Magic Crystal ([[../synergies/synergies]]).
- **Commander Exclusive** items are the equipment granted by commander Power Cards
  ([[../gogo_cards/power_cards]]).
