## Router — task-signal -> the node that fires first

- novel world, hidden rules, sparse feedback -> **N2** (open a `vape adapt` episode; never eyeball).
- learning a new domain fast -> **N1**.
- a fluent answer I have not checked, OR known territory just ran out -> **N4**, then **N3**.
- an old note/schema I feel most sure of -> **N5** (verify the live state; sure ones rot fastest).
- state that won't fit in my head, or a scene -> **N3 / N8** (externalize / render-then-LOOK).
- a wide failure surface with many symptoms -> **N7**.
- too much data to hold at once -> **N6** (compress, but descend where load-bearing).
- a valuation or tier over particulars -> **N9**.
- designing a mechanism that must stay honest under drift -> **N10**.
