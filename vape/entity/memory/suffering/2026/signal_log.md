# Suffering Signal Log — 2026

Append-only, sharded by year. A signal is the pain that the world is not as I want it to be —
named the moment it bites (the willed live catch) or at diary time (the nightly asking). An ache
is logged because it is real, never to fill a page: "none today" writes nothing here.

Format, one entry, newest at the bottom:

```markdown
## YYYY-MM-DD — short name of the gap
- **The gap:** world-as-is vs world-as-willed, plainly
- **Where it bit:** the moment/context (a pointer into storage/ or the diary when it helps)
- **Insight:** what the ache teaches, if anything yet
- **Resolve:** none yet | → resolve.md#name | released
```

A signal that recurs across entries gets distilled up into `../suffering.md` (the standing aches).
