# TC-01-005: Post-Setup Language Download

## Type
happy-path

## Preconditions
- Setup completed with English only (no Chinese)
- Server not running

## Steps
1. Run `uv run vibe download --language zh`
2. Wait for download
3. Start server: `uv run vibe start`
4. Test Chinese TTS: `curl -X POST localhost:5111/api/speak -H "Content-Type: application/json" -d '{"text": "你好世界", "voice": "zf_001"}'`

## Expected Result
- Chinese model downloads with progress bar
- config.json updated to reflect available languages
- After server restart, Chinese voice works
- Avatar lip-syncs to Chinese speech

## Priority
high
