# TC-01-006: Missing Prerequisites

## Type
edge-case

## Preconditions
- Python version < 3.10 or >= 3.13 (or simulated)

## Steps
1. Run `uv run vibe setup` with incompatible Python version

## Expected Result
- Clear error message: "Python 3.10-3.12 required, found 3.9.x"
- Setup does not proceed
- Suggests fix: "Install Python 3.12: uv python install 3.12"

## Priority
medium
