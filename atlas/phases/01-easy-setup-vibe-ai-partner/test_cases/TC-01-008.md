# TC-01-008: Node.js CLI Properly Archived

## Type
regression

## Preconditions
- Phase implementation complete

## Steps
1. Verify `apps/cli/` no longer exists
2. Verify `archives/cli-node/` contains the old CLI code
3. Verify root `package.json` no longer has `setup`, `start`, `stop` npm scripts
4. Verify `uv run vibe setup` and `uv run vibe start` are the documented entry points in README

## Expected Result
- `apps/cli/` moved to `archives/cli-node/` (preserved, not deleted)
- No npm scripts reference the old CLI
- README only documents `uv run vibe` commands
- No broken imports or references to the old CLI path

## Priority
medium
