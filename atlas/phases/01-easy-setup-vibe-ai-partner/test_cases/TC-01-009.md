# TC-01-009: Switch TTS Engine via Re-Setup

## Type
happy-path

## Preconditions
- Setup completed with Kokoro ONNX

## Steps
1. Run `uv run vibe setup` again
2. Select engine: KittenTTS (option 3)
3. Complete setup
4. Run `uv run vibe start`
5. Test TTS

## Expected Result
- KittenTTS deps install (Kokoro ONNX deps remain, not uninstalled)
- KittenTTS model downloads with progress bar
- config.json updated to `"ttsEngine": "kitten"`
- Server starts with KittenTTS engine active
- TTS works with KittenTTS voices

## Priority
medium
