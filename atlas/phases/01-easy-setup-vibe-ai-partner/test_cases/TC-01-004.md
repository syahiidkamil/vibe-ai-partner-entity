# TC-01-004: Skip-if-Cached — Re-run Setup Without Re-downloading

## Type
happy-path

## Preconditions
- Setup previously completed with Kokoro ONNX + Japanese
- Models exist in `~/.cache/kokoro-onnx/`

## Steps
1. Run `uv run vibe setup` again
2. Select same engine: Kokoro ONNX
3. Select same languages: Japanese

## Expected Result
- Model downloads are skipped: "kokoro-v1.0.onnx already cached (310MB) — skipping"
- UniDic download is skipped: "UniDic already installed — skipping"
- Setup completes in seconds (no network downloads)
- No re-download of any cached files

## Priority
critical
