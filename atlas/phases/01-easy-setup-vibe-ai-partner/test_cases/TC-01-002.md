# TC-01-002: Start Server — TTS + Avatar on Single Port

## Type
smoke

## Preconditions
- Setup completed (TC-01-001)
- Port 5111 is free

## Steps
1. Run `uv run vibe start`
2. Open browser to `http://localhost:5111`
3. Verify avatar renders (Live2D model visible)
4. Send a speak command via curl: `curl -X POST localhost:5111/api/speak -H "Content-Type: application/json" -d '{"text": "Hello world"}'`
5. Observe avatar lip-sync during speech
6. Press Ctrl+C to stop

## Expected Result
- Server starts in foreground with clear status messages
- No "downloading model" during start (already downloaded in setup)
- Avatar renders at `localhost:5111` (PixiJS + Live2D)
- TTS API responds on same port
- Lip-sync works via WebSocket (same-origin, no CORS)
- Ctrl+C gracefully shuts down server

## Priority
critical
