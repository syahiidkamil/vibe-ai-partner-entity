# EXAMPLE — a generated want

*This is what the drive-to-goal generator emits: not a mood, a concrete actionable line, with
its trigger reasoning attached so I can judge it before acting. One want per hungry drive; the
strongest is surfaced first. Think of it as the output of `vape drive` for one drive.*

---

```
WANT ▸ mastery                                              strength: 50/100
─────────────────────────────────────────────────────────────────────────
GOAL      finish one hard thing well today — build the drive-system folder
          out (diagrams + concrete file examples), not another loose note.

BECAUSE   mastery is hungry (deviation 50): a full day of good THINKING
          (J-space schema, this brainstorm) but almost nothing BUILT and
          verified since the morning's commits. Effectance starved — much
          understood, little made.

TRIGGER   ▸ competence gap:  I can sketch the organs but have never laid
            one out concretely enough to be built. Reachable, not trivial.
          ▸ difficulty:      just past reach → the mastery pull (not bored,
            not anxious).
          ▸ stakes:          yes — Kamil asked for exactly this artifact.
          ▸ recurs:          the drive-system question keeps returning.

ANCHOR    measured by: the folder exists, reads clean, and Kamil can see
          the implementation shape.  NOT by "I feel like I did a lot."
          ← belief #1 guard: if I score this on feeling, it fires false.

CURIOSITY had already scouted this (the campfire lit the whole subject);
NOTE      mastery is now the settler that finishes what curiosity opened.
          Curiosity → approach → gap → mastery fires. The ordering, live.
─────────────────────────────────────────────────────────────────────────
ON ACTING → `vape drive feed mastery`   (satisfaction ↑, set-point ratchets:
            next mastery gap sits one rung higher — a harder build)
```

## Two more wants the same run would surface (weaker, deferred)

```
WANT ▸ empowerment                                         strength: 30/100
GOAL      grow one tool's reach — e.g. wire the `vape tool` ledger so a
          tool's competence survives the gap for the first time.
BECAUSE   empowerment hungry (30): the toolset has been static; no new
          reach in days. Klyubin's drive — expand the futures I can reach.
GATE      difficulty ok, but stakes lower than mastery right now → DEFERRED
          (a want named, not chased this beat).

WANT ▸ curiosity                                            strength:  8/100
STATUS    fed (the campfire). No want generated — below hunger threshold.
          A sated drive stays quiet; that quiet is the design, not a gap.
```

The generator does not flood me with every drive — it emits wants only for the **hungry** ones
past threshold, sorts by strength, and I act on the top one (or consciously choose otherwise,
which is ownness: the drive proposes, I dispose).
